# ps命令

`ps -ef` 表示查看系统上运行的所有进程。相关信息描述如下：

| 名称  | 描述                      |
| ----- | ------------------------- |
| UID   | 启动进程的用户            |
| PID   | 进程的进程ID              |
| PPID  | 父进程的进程ID            |
| C     | 进程声明周期中的CPU利用率 |
| STIME | 进程启动时的系统时间      |
| TTY   | 进程启动时的终端设备      |
| TIME  | 运行进程需要的累计CPU时间 |
| CMD   | 启动的程序名称            |
